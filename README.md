# dind-with-compose
Sample codes for Docker-in-Docker with docker-compose.

## Quickstart

```
$ docker-compose up -d
$ docker-compose ps
$ docker-compose exec project-a-compose docker-compose ps
$ docker-compose exec project-b-compose docker-compose ps
```
